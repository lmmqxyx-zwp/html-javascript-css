# 工程结构(原生JavaScript)
```
.
├── css
│   ├── common (文件夹 | 主要的css样式)
│   └── main.css (主css样式)
├── doc (相关md文档)
├── img (相关图片)
├── js (web相关JavaScript)
│   ├── common (文件夹 | 主要的js文件)
│   ├── main.js (主要的js文件)
│   ├── plugins.js (相关插件)
│   └── vendor (第三方JavaScript)
│       ├── jquery.min.js (...)
│       └── modernizr.min.js (专为HTML5和CSS3开发的功能检测类库、html在浏览器中显示后、在`<html class="...">`的`class`上添加内容)
├── favicon.ico (图标)
├── index.html (主页文件)
├── .gitignore (git提交时相关配置)
├── .gitattributes (.gitignore参照属性)
├── LICENSE.txt (license文件)
└── *.html (被调用或者嵌入使用的html)
```
